Responsive web development (Mobile First)

Saker vi kan tänka på för att att få en hemsida att bli responsive

  • Media Queries

    Media queries är CSS-regler som tillåter oss att tillämpa stillmallar baserat på enehetens egenskaper, upplösning och orientering

  • Flexbox

    Flexbox gör det enklare att jobba med design som är mer responsive.

  • Responisiva bilder

    Bilder kan bete sig underligt men det finns ett enkelt knep

  • Responisiv typografering

    Det kan vara bra att se över sin typografering för olika media queries
